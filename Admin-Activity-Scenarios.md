# Privileged Account Risk Scenarios

## Scenario 1 – Late Night Admin Login

Risk:
Admin account used at unusual hours.

Impact:
Possible compromised credentials.

Monitoring:
Track login times and alert on anomalies.

---

## Scenario 2 – Multiple Failed Admin Logins

Risk:
Brute force attempt on admin account.

Impact:
Unauthorized system access.

Monitoring:
Check failed login attempts and lockout events.

---

## Scenario 3 – New Admin Account Created

Risk:
Privilege escalation attempt.

Impact:
Unauthorized high-level access.

Monitoring:
Track user creation events.

---

## Scenario 4 – Admin Accessing Critical Servers

Risk:
Potential misuse of privileges.

Monitoring:
Audit access to sensitive systems.
